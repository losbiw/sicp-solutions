
```scheme
(expand 1 7 10) ; (1, 4, 2, 8, 5, 7, 1, ...) - decimal representation of 1/7
```

```scheme
(expand 3 8 10) ; (3, 7, 5, ...)
```

The resulting stream is decimal expansion of the fraction $\frac{numerator}{denominator}$ (1-st and 2-nd arguments respectively), and the last argument is the base (?) 

The stream is infinite, so for a rational number (i.e. all possible inputs in this exercise) it will start yielding zeroes for infinity once the expansion is done

## Binary example

```scheme
(expand 1 7 2) ; (0, 0, 1, 0, 0, ...) - decimal representation of 1/7 in base 2
```
