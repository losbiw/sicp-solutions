
It works by drilling down to the bottom of the list first and returning an empty set (nil).

Then, it appends the current element to a copy of each of the already found subsets and combining them, like so.

(subsets (list 1 2 3))

(())
(() (3))
(() (3) (2) (2 3))
(() (3) (2) (2 3) (1) (1 3) (1 2) (1 2 3))
