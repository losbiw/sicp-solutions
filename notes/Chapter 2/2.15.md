Overall, the uncertainty of the outcome should decrease if we decrease the total uncertainty from each individual component of the equation. Thus, if the uncertain values (r1 and r2) are reduced to as few uses within the equation as possible, the outcome should be more certain than if they appeared more than once.

Thus, par2 should be more suited for calculating parallel resistance when resistances are uncertain (have a non-zero tolerance)

The system currently has no way of knowing when it's dealing with the same interval twice, so it combines uncertainties as if they came from separate measurements. Can be seen in dividing an interval by itself.
