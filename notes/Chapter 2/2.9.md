
interval a: [a1, a2]
interval b: [b1, b2]

Width a = (a2 - a1)/2
Width b = (b2 - b1)/2

### Addition:
a + b = [a1 + b1, a2 + b2]
Width(a + b) = (a2 + b2 - a1 - b1)/2 = (a2 - a1)/2 + (b2 - b1)/2

### Subtraction
a - b = [a1 - b2, a2 - b1]
Width(a - b) = (a2 - b1 - a1 + b2)/2 = ((a2 - a1) + (b2 - b1))/2 = (a2 - a1)/2 + (b2 - b1)/2

### Multiplication/division

Counterexample: [3.35, 3.65] and [4.25, 4.75]
