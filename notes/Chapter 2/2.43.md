Normal order:
1. Find all combinations for k-1 queens
2. Filter them (implied in recursive structure)
3. For each existing k-1 combination, add k-th queen to each available row
4. Filter by safety

Louis Reasoner order:
1. Find all ways of placing k-th queen in current row
2. For each way to place k-th queen, find all ways to place k-1 queens and filter them by safety
3. Filter final combinations by safety

The second version of the program redundantly searches for all ways to place k-1 queens for each way to place the kth queen. I.e, instead of being called k-1 times in first example.

For each queens-col call in original version, Louis's version calls it an extra k times (one for each row). Therefore, the time it takes will be roughly $T*k^k$
